URL Parameters

It is possible to add the IP address to the URL to go directly to that IP and calculate it. Each of the following URLs will work:

/tools/subnet-calculator/?ip=192.168.1.1/24
/tools/subnet-calculator/?ip=10.7.7.0 255.255.255.0
/tools/subnet-calculator/?ip=2001:0db8:85a3::8a2e:0370:7334/64

This allows for using keyword searches in Firefox.
Duckduckgo.com users can enjoy the !subnet bang command which makes use of this tool. Try !subnet 192.168.1.1/24.

About this Subnet Calculator

This free online subnet calculator allows for the input of a netmask, a Cisco wildcard mask or /CIDR notation.
The tool also has the ability to create a random IP address which can then be calculated. This is useful when studying and learning how to subnet and example practice networks are needed.
This calculator can also be used to do reverse subnetting where the number of hosts is known but the netmask is not.
Calculator is also IPv6 ready.
